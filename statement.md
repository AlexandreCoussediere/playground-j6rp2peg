3. Core Concepts
A. Variables and Types
Python uses dynamic typing, meaning you don't have to explicitly declare the type of a variable.
```name = "Alex"          # String
age = 30               # Integer
price = 19.99          # Float
is_learning = True     # Boolean```
B. Indentation
Unlike other languages that use curly braces {} to define blocks of code, Python uses indentation (whitespace). This forces you to write clean, organized code.
```if age > 18:
    print("Access granted.")
else:
    print("Access denied.")```
